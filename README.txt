如何把內建示意圖換成「專業圖檔」
=====================================

本頁每個工具的設計圖預設用內建向量圖(SVG)。若你取得更精美的圖
(原廠衛教插圖、委託設計的醫學插圖、或已授權的圖庫圖),可直接替換:

1. 把圖片(建議 PNG 或 SVG,寬約 600–1200px)放進本 assets/ 資料夾,
   例如: assets/dims_front.png

2. 打開 index.html,找到 const DIAGRAMS={ ... }
   在對應那張圖的物件裡加一個 img 欄位,指到你的檔案:

   之前:
     {cap:"DIMS · 正面(蜂巢)", svg:`<svg ...>...</svg>`, notes:[...]}

   之後(加 img,會優先用圖檔、忽略 svg):
     {cap:"DIMS · 正面(蜂巢)", img:"assets/dims_front.png", svg:`<svg ...>...</svg>`, notes:[...]}

3. 存檔、重新上傳 index.html 即可。右側的編號說明(notes)不受影響,
   會繼續顯示;點圖一樣可放大。

注意:
- 使用原廠或圖庫圖片前,請確認你有合法使用/授權(醫療器材行銷另受法規規範)。
- 若只想換部分圖,其餘沒加 img 的會繼續用內建 SVG。
